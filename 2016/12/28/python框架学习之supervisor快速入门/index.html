python框架学习之supervisor快速入门

发表于 2016-12-28 | 分类于 python ， python框架学习 |

前言

我们在日常做Linux环境部署的时候经常需要守护进程或者做进程统一管理，如果觉得自己写脚本太麻，或者不知道如何来管理，那很可能supervisor正好是你所需要的

supervisor 简介

supervisor 是用Python开发的一套通用的进程管理程序，能将一个普通的命令行进程变为后台daemon，并监控进程状态，异常退出时能自动重启。 supervisor 主要由 supervisorctl, supervisord, echo_supervisord_conf 三部分组成:

  • echo_supervisord_conf: 主要负责配置文件的生成
  • supervisord: 相当于守护进程，主要对进程做管理，启动supervisor主进程，启动所管理的子进程以及响应supervisorctl的命令对子进程进行管理,包括，启动，停止，重启子进程，等等
  • supervisorctl: 是客户端工具，用它可以和supervisord进行通信，方便客户端管理员手工对进程进行操作
  • web service: 提供web端管理界面方便查看（暂不介绍）

Mac 环境下 supervisor 安装

easy_install supervisorysq 或者 pip install supervisor

supervisor 配置文件

配置文件生成

安装好supervisor后默认是没有配置文件的，需要我们手工生成，命令如下：

echo_supervisord_conf > /etc/supervisord.conf

生成的配置文件主要用于supervisord程序运行时加载配置项

配置文件功能详情

supervisord 启动

  • supervisord 启动配置文件加载路径优先级
  • supervisord 默认去找 $CWD/supervisord.conf，也就是当前目录
  • supervisord 默认 $CWD/etc/supervisord.conf，也就当前目录下的etc目录
  • supervisord 默认去找 /etc/supervisord.conf的配置文件
  • supervisord -c /home/supervisord.conf 其中 -c 是 config 的意思，从指定路径加载配置文件

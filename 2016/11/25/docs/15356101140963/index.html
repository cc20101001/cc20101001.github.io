清华大学: https://pypi.tuna.tsinghua.edu.cn/simple/

中国科学技术大学: http://pypi.mirrors.ustc.edu.cn/simple/

使用方法:

使用方法很简单，直接 -i 加 url 即可！如下(安装web.py)：

pip -i http://pypi.douban.com/simple install web.py

如果有报错,请使用命令：

pip -i http://pypi.douban.com/simple install web.py --trusted-host pypi.douban.com

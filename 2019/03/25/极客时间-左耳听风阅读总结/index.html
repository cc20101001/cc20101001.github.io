发表于 2019-03-25 | 分类于 书 ， 极客时间 |

前言

看了下左耳朵耗子的左耳听风，耗子叔才华横溢，思维天马行空，小弟跟不上，不得不总结记录一点东西，方便日后回顾（一时不会回顾吧-_-），免得走马观花，总结的原则也是简单起见，每一篇文章最少需要一句话，也就是中心思想，或者自己喜欢的一句就行（耗子叔总想表达的非常优雅非常完美，所以表现出来很啰嗦）

10.如何成为一个大家愿意追随的 Leader

明白 Leader 和 Boss 的区别，Leader 是大家一起做事相互帮助解决问题的互助式关系，Boss 更偏向于命令式，指令式，要你干啥就干啥，清楚自己的定位，办起事情来就会顺畅很多

11.程序中的错误处理-错误返回码和异常捕获

耗子很详细的方法面面的分析了各种编程语言异常处理方便的优劣，我的收获其实没有那么大，觉得他分析的有点片面，实际工作工程中往往是根据自己的编程语言特性，多种异常处理方式相结合的办法，比如 python，我们使用的就是 try，except 相结合，并且里面结合的自己包装的错误码机制，是我开发这些年碰到的 python 中最好的处理方式，看起来非常清晰

12.程序中的错误处理-异步编程

主要讲了 js 中异步编程怎么去处理错误，注册回调函数如，onerror onsuccess

前言

这个系列主要介绍lua语言学习，某些地方也会和同为脚本语言的python进行对比，第一节里主要包含了lua一些基本知识

1.环境相关

2.注释

3.数据类型，作用域

4.逻辑运算，运算符

5.数据类型

6.迭代器，表

交互式编程模式

lua提供了交互式编程模式，我们在命令行中用 lua -i 命令或 lua，就可以使用了（同python）

脚本式编程模式

lua也提供了脚本式编程模式，我们写一段lua代码，放在 .lua 结尾的文件中，这个 .lua文件就是一段lua脚本文本

如：test.lua

我们如果想运行这段脚本只需要输入 lua(luajit) test.lua 即可

注释

lua里面的注释分两种，行注释和代码块注释

行注释

在lua里面用 -- 表示行注释，同python里面的 #

代码块注释

在lua里面代码块注释用 --[[ *** ]]-- 表示代码块注释，同python里面的 “”“ *** ”“”（如果非要扣细节，python里面没有多行注释）

数据类型

lua和python一样，都是弱类型语言（也可以叫动态语言），变量不需要类型定义，只需要赋值，你赋值为什么类型，他就会是什么类型，在lua中有八种基本类型分别为: nil、boolean、number、string、userdata、function、thread和table

数据类型 描述
nil 这种为null，又如一个没有被赋值的变量，也可以被用来删除某个变量
boolean bool类型，Lua 把 false 和 nil 看作是”假”，其他的都为”真”
number lua只有一种数字类型，双精度类型的实浮点数
string 字符串类型，可以用一对双引号或者单引号来表示，也可以用”[[]]” 来表示”一块”字符串，和python里面的 “”” 字符串 “””类似
userdata 这种为null
function 函数类型
thread 函数类型
table 实际是个”关联数组”（associative arrays），数组的索引可以是数字或者是字符串，并且索引从1开始

变量和作用域

lua里面的变量分全局变量和局部变量，全局变量的作用域为当前lua文件，局部变量用local开头，作用域只在当前代码块，正常写代码的时候我们尽量使用局部变量，因为局部变量访问速度比全局变量要快

循环

lua的循环语句支持for循环，while循环，repeat…until（这个有点类似于do while语句）以及循环嵌套

逻辑判断

逻辑判断和其他语言并无不同，支持if语句，支持if else，支持if语句嵌套

函数

lua对函数支持有一点比较特别

1.他的函数有全局函数和局部函数之分，这点和python有点不一样

2.他也支持返回多个值

3.他也支持多参数（不定长参数列表）

lua字符串

lua语言中的字符串表示形式有三种

1.单引号间的一串字符

2.双引号间的一串字符

3.[[和]]间的一串字符（这个和python里面的”””和”””中的一串字符比较像，还可以换行）

在lua里面还支持转义字符，和其他语言类似，都是用 \， 用于表示不能直接显示的字符，比如后退键，回车键等，再比如在”\””就会输出一个双引号的字符串

字符串操作

lua里面还提供了很多的方法来支持字符串的操作

方法 用途
string.upper(argument) 字符串全部转化为大写字母
string.lower(argument) 字符串全部转化为小写字母
string.gsub(mainString，findString，replaceString，num) 在字符串中替换，mainString为要替换的字符串，findString为被替换的字符串，replaceString为要替换为的字符串，num是替换次数（如果忽略则全部替换）
string.find(str，substr，[init, [end]]) 在一个指定的目标字符串中搜索指定的内容，并返回其具体的位置，如果没匹配到则返回nil
string.reverse(arg) 字符串反转
string.format(…) 格式化字符串，如string.format(“the value is: %d”， 4)
string.char(arg)和string.byte(arg[,init]) char将整型数字转成字符并连接，byte转换字符为整数值
string.len(arg) 计算字符串的长度，也可以用 # 一元运算符
string.rep(string, n) 返回字符串string的n个拷贝
.. 连接两个字符串
string.gmatch(str, pattern) 返回一个迭代器函数，每次调用，返回一个在字符串str找到的下一个符合pattern描述的子串。如果参数pattern描述的字符串没有找到，迭代函数返回nil

迭代器

迭代器是一种对象，它能够用来遍历标准模版库容器中的部分或全部元素，每个迭代器对象代表容器中的确定的地址 在lua中迭代器是一种支持指针类型的结构，它可以遍历集合的每一个元素

如ipairs函数，就是lua内置的一个迭代器函数，可以返回一个迭代器，用来遍历数组

ipairs和pairs的区别

ipairs 仅仅遍历值，按照索引升序遍历（key如果为非索引值则不遍历），索引中断停止遍历。即不能返回 nil,只能返回数字 0，如果遇到 nil 则退出。它只能遍历到集合中出现的第一个key不是整数的对象。

pairs 能遍历集合的所有元素。即 pairs 可以遍历集合中所有的 key，并且除了迭代器本身以及遍历表本身还可以返回 nil。

表

table是lua的一种数据结构用来帮助我们创建不同的数据类型，如数字，字典等

lua table使用关联型数组，你可以用任意类型的值来作数组的索引，但这个值不能是 nil，table的大小是不固定的，可以根据自己的需要进行扩容

table常用操作

方法 用途
table.concat 字符串拼接
table.insert 插入元素
table.remove 移除某个元素
table.sort 对给定的table进行升序排序

前言

先说明 jetbrains 破解工具不是我开发的，我只是搭建了一个破解 jetbrains license 的 web services，方便大家可以用，地址如下：http://jetbrains.hubinwei.me/，不客气

jetbrains破解方法

我们在使用 jetbrains 的时候精彩会碰到软件授权问题，本文档方便记录 jetbrains 破解方法，方便自己日后查看，jetbrains破解方式有三种:

1.license授权码

2.license服务器授

3.更改系统时间授权

这里介绍方法2，并且包括了如何搭建自己的 license 服务器

2、找到自己操作系统的版本，我这里使用IntelliJIDEALicenseServer_windows_amd64.exe，双击打开运行

3、在idea注册界面选择授权服务器，填写 http://127.0.0.1:1017，然后点击“OK“

坑多多

1.如果通过授权码授权，局域网里面如果两台机器使用相同的授权码，其中有一台机器会被判定为异常，授权失效

2.授权服务器授权方式最好，因为一台服务器可以授权所有的 jetbrains 软件

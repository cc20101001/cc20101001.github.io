python 里面 __name__, __main__, sys.argv介绍

__name__ 是 python内置常量，常用于区别当前的模块的执行方式，如果你是直接运行该文件，他的值就是缺省值： __main__，如果你是通过的 import 导入的该模块，那么他的值就是当前模块的文件名

sys.argv python传参

我们在自己写python代码的时候经常需要获取外部参数，在python里面是用sys模块的argv列表对象来接收的，比如我们执行 python test.py 123 456 789 sys.argv获取到了四个参数，依次为:

python表示用python解释器去执行test.py文件,后面的四个参数依次会当作外部参数传给python解释器，并存储在sys.argv列表对象里面（数据类型为list）

python 里面的比较

python里面的比较最常见的有 is 和 =，其中 is 是内存的比较，= 是值比较，因此常用 is 关键字比较速度更快

python 对象内存判断

如果需要定位python里面的某个对象的内存，常用的是调用内建函数 id()，显示的是十进制内存地址，也可以 print 对象，里面会显示十六进制的内存地址

python -m 参数说明

启动 demo.py 文件有两种启动方式

我们平时用的最多的是方式一，即立刻运行方式，还有另外一种启动方式是模块启动方式，相当于先把文件当作模块导入，然后再启动，他们两种启动方式的区别在于 sys.path，方式一会把当前路径添加到环境变量里面，而方法二不会把当前路径添加到环境变量

time 模块

time.sleep(s) 线程推迟指定的时间运行，单位为秒

time.clock() 这个需要注意，在不同的系统上含义不同。在UNIX系统上，它返回的是“进程时间”，它是用秒表示的浮点数（时间戳）。而在WINDOWS中，第一次调用，返回的是进程运行的实际时间。而第二次之后的调用是自第一次调用以后到现在的运行时间。（实际上是以WIN32上QueryPerformanceCounter()为基础，它比毫秒表示更为精确）

time.time() 返回当前时间的时间戳

time.localtime([secs])：将一个时间戳转换为当前时区的struct_time

time.localtime(1304575584.1361799) time.struct_time(tm_year=2011, tm_mon=5, tm_mday=5, tm_hour=14, tm_min=6, tm_sec=24, tm_wday=3, tm_yday=125, tm_isdst=0)

datetime 模块

在 python 中，我们经常需要用到时间比较，python 的 datetime 模块可以很方便帮我们解决问题

t_1 = (datetime.datetime.now() + datetime.timedelta(seconds=10)).strftime(“%Y-%m-%d %H:%M:%S”)

t_2 = datetime.datetime.now().strftime(“%Y-%m-%d %H:%M:%S”)

比较方法非常的简单，直接：t_2 > t_1，实际上用到的就是字符串比较如（”2018-03-28 14:26:47” > “2018-03-28 14:26:46”）

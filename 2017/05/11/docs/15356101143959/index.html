SCSS，SASS学习笔记

发表于 2017-05-11 | 分类于 前端 ， css |

前言

前端技术发展太快，之前的CSS语法显得很拖沓，臃肿，不方便管理，于是一部分人想对SASS语法进行升级和改进，于是CSS预处理器应运而生。CSS预处理器是一种新的编程语言，编译后生成正常的CSS文件。为CSS增加一些编程的特性，无需考虑浏览器的兼容问题，让CSS更加简洁，适应性更强，可读性更佳，更易于代码的维护等诸多好处，让我们开始撸一撸SCSS吧。

什么SCSS

我们通常说的SCSS就是一种CSS预处理器（可以理解为一套CSS高级语法），同类产品还有SASS、LESS等，这里我们重点介绍SCSS，他和原CSS语法最像

SCSS，SASS是什么关系

SCSS，SASS都是工具SASS支持的CSS预处理器，（这个工具使用ruby语言编写需要用gem安装）

区别:

文件扩展名不同：“.sass”和“.scss”

SASS是以严格缩进式语法规则来书写的，不带大括号和分号；而SCSS的语法和CSS书写语法类似

SASS的安装

MAC用户安装非常方便 gem install sass

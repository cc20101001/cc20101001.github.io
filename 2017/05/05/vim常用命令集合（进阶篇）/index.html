vim常用命令集合（进阶篇）

发表于 2017-05-05 | 分类于 vim |

前言

这里针对我自己收集了一份vim进阶命令，过段时间我就会忘记，所以要多加练习

光标移动

光标上下左右移动:

小写 h，j，k，l，来分别控制光标的左，下，上，右

左右移动:

shift + ^，光标移动到光标所在行最左侧 shift + $，光标移动到光标所在行最右侧

w，光标快速移到下一个字符的开头 b，光标快速移动上一个字符串开头

#l，光标往后移动＃个位置

翻页

ctrl + b 往回翻一页，ctrl + f 往后翻一页

删除

X，每按一次，删除光标所在位置的前面一个字符

10x，表示删除光标所在位置的后面十个字符

复制

4yy，复制包括当前行所在的四行到缓冲区

行跳转

4gg 或者 4G，表示跳转到第四行 gg，表示跳转到首行 G，表示跳转到尾行

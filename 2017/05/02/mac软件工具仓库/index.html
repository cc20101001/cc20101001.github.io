mac软件工具仓库

发表于 2017-05-02 | 分类于 mac ， 软件，工具 |

前言

这里收集整理了一些我常用的mac软件，工具，方便重装系统的时候用得上

装机必备

终端工具

zsh，iterm 等

iterm 终端显示错误问题 ctrl(还是commond) + ]

翻译工具

欧路词典

苹果下的翻译软件推荐两款，欧路词典是首选，其次是有道词典。

软件界面如下:

文档阅读工具

clearview

1.只需把存放图书文件的文件夹从 Finder 中拖拽到 Clearview 的书架，图书文件信息将被迅速导入，然后您就可以方便地浏览，搜索，组织，和阅读图书了。通过图书文件夹和阅读列表结合的方式来管理图书，一切都变得简单起来。

2.支持 PDF, EPUB, CHM, MOBI 四种流行的图书格式，Clearview 对这几种格式的图书都提供了统一的上佳阅读体验。

PDF Expert

极致的 PDF 阅读标注工作体验，专业的pdf阅读工具

Dash

Dash是一个API文档浏览器（ API Documentation Browser），以及代码片段管理工具（Code Snippet Manager）。它就只有这两个功能，但确实是程序员最为关心的特性，可以毫不夸张地说，Dash是它们之中做的最好的一个！非常实用。从它第一版发布用到现在，绝对是一个你值得拥有的文档管理工具.

记录笔记工具

MWEB

MWeb 是一站式的 Markdown 编辑和静态网站生成解决方案，完美的markdown工具

有道云笔记

特点是支持多系统共享笔记，缺点是mac版对团队协作支持的还不够好，需要用他的网页版

软件架构思维图

monodraw

Monodraw是一款专为 Mac OS X 设计的强大的ASCII艺术编辑器，纯文本历经几十年而不衰。monodraw可以创建基于文本的艺术图，布局，流程图，以直观地表示算法，数据结构，二进制格式和更多的东西。因为这一切都只是文本，所以它可以很容易地嵌入在几乎任何地方。

MindNode

Mindnode Mac版是Mac平台上的一款Mac思维导图软件，Mindnode Mac版能够让我们快速的创建出美观的思维导图，单击一次即可创建新节点，支持iCloud共享，并且具有iPhone和iPad客户端，支持 iCloud Drive 和 HandsOff 协作功能

EDraw Max

亿图图示专家(EDraw Max)是一款基于矢量的绘图工具，包含大量的事例库和模板库。可以很方便的绘制各种专业的业务流程图、组织结构图、商业图表、程序流程图、数据流程图、工程管理图、软件设计图、网络拓扑图等等。它帮助您更方便，更快捷的阐述设计思想，创作灵感。在设计时它采用全拖曳式操作，结合4600多个常用图形模板库和用户自定义实例库，最大程度简化用户的工作量；让你在设计时既可以充分利用固有的素材，又可以借鉴他人的作品。

水印相关

Watermarker for Mac

Watermarker for Mac是一款Mac平台的图像水印添加工具，Watermarker for Mac是一个非常专业的图形水印添加工具，能够帮助我们非常方便的给图像添加指定的水印，支持批量操作，非常方便和使用的图像水印添加工具。

Watermark Plus

这是一款Mac平台的超强图像水印批量添加工具，能够帮助我们用户给图片批量添加水印，添加的水印适合任何尺寸的大小，添加完之后还可以批量的修改图片的名称，保存为自己想要的图片格式。

http抓包发包相关

PostMan

超级强大，适合做自动化测试

Paw

Paw 是一款Mac上实用的HTTP/REST服务测试工具，完美兼容最新的OS X El Capitan系统，Paw可以让Web开发者设置各种请求Header和参数，模拟发送HTTP请求，测试响应数据，支持OAuth, HTTP Basic Auth, Cookies，JSONP等，这对于开发Web服务的应用很有帮助，非常实用的一款Web开发辅助工具。

Charles

Charles是一款用于HTTP信息抓包工具，可以快速有效的获得HTTP信息，非常利于开发者的网页开发和调试修改等！Charles 有着可视化的操作界面，非常利用编辑者的使用和调试！

wireshark

上古报文抓取分析工具

git相关

Kaleidoscope

Kaleidoscope 是一款很强大的文件和图像比较工具，支持任何文本文件和 jpeg、tiff、png、psd 等格式的图像文件，同时和 git、svn 等版本控制工具能够完美的结合，mac下最好用的diff compare工具

sourcetree

版本控制工具

deffmerge

mac下 diff compare工具，还行，但是不如Kaleidoscope

照片相关

Lyn

Mac下一款轻量级快速查阅图像软件

下载工具

迅雷

同windows

webtorrent

非常好用一款下载种子下载器，下载国外资源非常快

开发效率提升工具

Alfred

Mac下出名的工作流工具

mac终端设置代理

发表于 2017-05-03 | 分类于 mac |

前言

在mac终端经常需要使用一些命令来下载外网的东西，或者更新程序（brew，pip，npm等等），会非常慢，我联想到能否给终端配置代理设置呢，一查果然可以。

原理

把代理服务器地址写入shell配置文件.bashrc或者.zshrc（普通shell用.bashrc，zsh用zshrc），让代理服务器随程序加载而启用

方法2

设置代理

在命令行输入如下格式命令，会给终端设置http和https代理

export http_proxy=http://127.0.0.1:1087;export https_proxy=http://127.0.0.1:1087;

取消代理

export http_proxy=””;export https_proxy=””; 或者 unset http_proxy

前者是给代理赋值为空，后者为移除环境变量里的某个变量

angularjs1.x学习笔记

发表于 2017-05-10 | 分类于 js ， angularjs |

前言

普通js可通过 <script> 标签添加js代码到 HTML 页面。

AngularJS 是一个 JavaScript 框架，它通过指令扩展了HTML，且通过表达式绑定数据到 HTML，让前端html和js的结合更灵活，功能更强大。

以前接触过一点AngularJS的开发，每次都只是开发某个小模块，没有系统的学过，现在有一整块时间可以来用来好好研究一下AngularJS，想重新理一下，让思路更清晰一点

学习前的准备

要学好angularjs需要准备一些css，javascript，html，dom相关知识，这些是基础，如果不熟悉也没关系，可以边看边学

AngularJS 表达式

我们将AngularJS表达式写在双大括号 里面，表达式里面的语法和js语法有点像，可以自己定义变量，进行变量数学运算，字符串运算，bool运算，等等，和 ng-bind 指令有异曲同工之妙，但他更灵活，更方便的嵌入到 html 中。

AngularJS指令

AngularJS指令算是AngularJS非常核心的一个篇章了，我们绝大多数工作都会和这些指令打交道，内置指令都以ng-开头，都可以在angularjs应用程序中被调用，这里我简单介绍几个指令

指令名 用途 补充说明
ng-app 定义应用程序的根元素。
ng-init 初始化应用程序参数 一般不用这种方式初始化
ng-model 指令用于绑定控制器数据到 HTML(input, select, textarea)
ng-bind 绑定应用程序数据到 HTML 元素
ng-click 定义元素被点击时的行为
ng-repeat 用来循环数组
ng-show 显示或隐藏 HTML 元素

AngularJS模型

模型就是model的意思，是angularjs中的数据层，模型最主要介绍ng-model指令，他是AngularJS双向绑定的关键

其中 angular.module 用来初始化一个应用程序，ng-app="myApp" 用于将初始化的应用程序绑定到某个 dom 节点，$scope.name = "John Doe"; 定义了一个变量 name，然后使用 ng-model 就可以把变量 name 绑定到html（这里是input），这是把model层数据绑定到应用程序，同事应用层数据变化也会影响model层数据的变化，这就是AngularJS的双向绑定。

AngularJS过滤器

过滤器简介

AngularJS过滤器可以用来转换数据，调用非常简单，有点类似于Linux里面的管道，也是一个管道符号（|）

uppercase 过滤器会把当前作用域中的lastName变量转化为大写字符，然后输出到html页面。

内置过滤器列表

过滤器 描述 ｜ 补充
currency 格式化数字为货币格式。 字符串格式化
filter 从数组项中选择一个子集。 字符串过滤
lowercase 格式化字符串为小写。
orderBy 根据某个表达式排列数组。 根据某个字符进行排序
uppercase 格式化字符串为大写。

过滤器实战

自定义过滤器（返回的是一个函数）

自定义过滤器依赖于 .filter 语法，里面接受两个参数，一个是定义的过滤器名称，另一个是过滤函数，这个函数是一个闭包，返回的是一个函数（像python里的装饰器），并且支持注入依赖

AngularJS服务

在AngularJS中服务分为内建服务（30多个）和自定义服务例如有一个 $location 服务，可以返回当前url

location服务

调用AngularJS的 $location 服务（相当于deom的window.location服务，但是兼容性更好），获取当前页面的绝对路径，并且在前端页面呈现:

http服务

AngularJS还有一个核心服务是 $http，

注意：$http服务默认是不能跨域的

timeout服务

$timeout 服务类比 window.setTimeout，表示延时执行，可以用于异步处理

interval服务

$interval 服务对应了 window.setInterval 函数，是一个定时触发器，表示隔一段时间就执行某个函数

自定义filter，service混合使用

AngularJS select下拉选项

AngularJS针对select下拉选项有一个专门的指令，ng-options，我们其实也可以用ng-repeat来解决，但是效果不如ng-options正统

ng-options个人建议数据格式是列表，然后列表里面是（key，value）对

AngularJS表格

AngularJS对表格的支持挺好（还不是一个ng-repeat），这个比较简单，直接上example就可以

AngularJS的DOM支持

主要为三个指令，也挺简单

指令名 作用
ng-disabled 直接绑定应用程序数据到 HTML 的 disabled 属性
ng-show 隐藏或显示一个 HTML 元素
ng-hide 用于隐藏或显示 HTML 元素

AngularJS事件

如 ng-click，等类DOM事件

AngularJS表单

AngularJS对表单的支持，其实就是控制器对表单的支持很棒，ng-model能够很好的和表单控件进行双向绑定

AngularJS内置函数

AngularJS提供了一些常用的内置函数，如:

函数名 用法
angular.lowercase() 转换字符串为小写
angular.uppercase() 转换字符串为大写
angular.isString() 判断给定的对象是否为字符串，如果是返回 true。
angular.isNumber() 判断给定的对象是否为数字，如果是返回 true。

AngularJS包含

包含网页

AngularJS中使用 ng-include 指令，能够在html中include其他html页面

include还可以包含其它js代码

跨域访问

默认情况下， ng-include 指令不允许包含其他域名的文件。如果你需要包含其他域名的文件，你需要设置域名访问白名单（使用.config来配置）

此外，还需要设置服务端允许跨域访问

前言

在文章 python模块分析之threading-基础 中我们介绍了线程的基本概念和线程的启动，在二中我们将介绍线程间数据的传输

线程间数据传输之使用队列 Queue

官方说明：创建一定大小的队列，默认是maximum为0，为无限大

用queue在多个线程中传递数据，是线程安全的，我们在对他做操作的时候已经包含了必要的锁，因此不必担心多个线程同时操作一个对象存在资源竞争问题

核心操作函数：

  1. Queue.get() 出队，还可以设置 block 或者 timeout，避免永久阻塞

  2. Queue.put(value) 入队，同 get 也可以设置 block 或者 timeout

Queue还有 qsize()，full()，empty() 方法，来获取队列当前状态，但是这些方法不是线程安全的，你现在获取了一个状态，可能立刻有元素新入队，就有问题了，使用时候需要小心

有了这些api接口，然后多个线程共享Queue对象，是不是操作非常简单？

值得注意的一点是，我们使用简单队列在线程间传递数据，实际上是传递的对象的引用，因此如果你担心数据共享问题可以传递深拷贝对象或者传递一些不可改变的对象类型，如 string，tuple

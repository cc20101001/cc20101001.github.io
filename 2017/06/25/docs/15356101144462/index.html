macbook pro硬件升级完美攻略

发表于 2017-06-25 | 分类于 mac |

前言

本人有一款2011年初产的macbook pro（MC724LL/A），具体配置如下：

显示器大小: 13.3寸
硬盘: 500GB，HDD
内存: 4G
cpu: 2.7 GHz Intel Core i7

最近想把电脑升级一下，把内存升级到8G，把光驱换为硬盘架，然后把硬盘升级到620GB（500GB + 120GB）

笔记本型号确认

想查看自己苹果笔记本型号的请看这里 http://www.everymac.com/systems/apple/macbook_pro/index-macbookpro.html

想查看自己苹果电脑能否升级硬件，请看这里 https://support.apple.com/kb/SP619?locale=zh_CN&viewlocale=zh_CN

工具包

笔记本型号看好了，开工起嘛需要工具吧，建议去某宝买个工具箱套装，价格在20块钱以内，里面工具非常多，可以满足日常拆机需求，工具图如下

升级内存

内存升级相对比较简单，先查看自己笔记本主板是否支持内存扩容，以及最大支持容量。然后看内存的型号，高型号是兼容低型号的，如本人原内存是4G DDR3 1333，新购买了一根4G DDR3 1600，打算混用，扩容为8G

升级硬盘

升级硬盘就比较麻烦了，况且我还需要买硬盘架，大体分为三部分

1.确定硬盘尺寸

2.确定SATA类型

3.确定硬盘架尺寸

4.拆机更换硬盘架，更换磁盘

5.系统克隆

6.我的操作步骤

硬盘尺寸

主流硬盘大小是2.5和3.5寸，一般来说笔记本都是2.5，服务器是3.5，所以买2.5，不必纠结

SATA类型

SATA类型分你主板支持的类型，你外置硬件设备的类型，链接速度为6千兆位的是SATA3主板，如果协商的链接速度也是6千兆位，那么你磁盘也是SATA3接口。如果协商的链接速度是3千兆位，那么你的磁盘是SATA2接口（SATA2的磁盘和SATA3的主板的协商链接速度只有3千兆位），这里也果断买SATA3磁盘吧

硬盘架尺寸

主流硬盘架尺寸分两种，一种是12.7mm，一种是9.5mm，苹果电脑都是9.5，所以买这个尺寸就对了。（其他电脑很多都是12.7）

拆机

这个算是我们一个核心步骤，其实也很简单，我直接给个教程吧

http://bbs.feng.com/read-htm-tid-4265042.html

系统克隆

硬盘选好了，总不至于给新硬盘重新安装一个系统吧，这里推荐mac的 Carbon copy cloner，支持硬盘级别的克隆，我们只需要把两块硬盘同时接到笔记本，完成数据拷贝即可，拆下原系统盘，然后重启，就可以进入新系统了，克隆界面如下：

我的操作步骤

我实施的具体步骤如下:

1.换下光驱为硬盘架＋500G SSD

2.克隆系统从500G HDD到500G SSD

3.安装500G SSD到原硬盘位，安装另一块120G SSD到光驱位，原来的500G HDD不要了

mac电脑开机优化及磁盘优化

初次开机可能非常慢，因为mac主板上NVRAM信息不是最新的，需要我们来手工更新一下，电脑开机时同时按 Command+Option+P+R 重置主板NVRAM，然后有需要再安装 Trim Enabler 优化下ssd性能，新硬盘完美替换，最后来张效果图-_-！

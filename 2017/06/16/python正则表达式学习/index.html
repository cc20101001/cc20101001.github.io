python正则表达式学习

发表于 2017-06-16 | 分类于 正则表达式 |

前言

正则表达式不是 python 所独有的，而是所有编程语言有相关模块支持的，但是它们的基本思想是相同的，本篇文章将主要围绕 python 进行正则表达式的学习

在 python1.5 之前 python 用的 Emacs 风格的正则表达式，在1.5引入了 re 模块，使用 perl 风格的正则表达式，你可以把正则表达式看作一门小型，专业度非常高的编程语言，他有内置的字符集和语法，专门来对字符串进行处理

普通字符

在正则表达式里面都是用提供的字符去和待求值字符串进行比较，我主要围绕我们提供的字符进行分析

一般字符

语法 说明 表达式实例 想匹配的字符串
一般字符 匹配自身 abc abc
. 匹配所有除了换行符\n外的所有字符 a.c abc
\ 转移字符，把内置字符转化为某个变量 a.c a.c
[…] 字符集，在字符集中所有特殊字符失去含义变为普通字符串，在字符集中有^表示取反 a[123]c a1c或a2c或a3c，在字符集中可以使用其他字符集

预定义字符集

这个字符集是我们使用最多的一些字符集，在字符集[…]中也可以使用

语法 说明 表达式实例 可以匹配到的字符串
一般字符 匹配自身 abc abc
\d 数字[0-9] a\dc a1c等等
\D 非数字[^\d] a\Dc abc
\w 字母活着数字[a-zA-Z0-9] a\wc abc
\W 非字母且非数字[^\w] a\Wc a*c
\s 空白字符[空格\t\n\r\f\v] a\sc a c
\S 非空白字符[^\s] a\Sc abc
\t tab
\r 回车
\v 竖向制表符
\f 换页符
\n 换行符

数量词

可以写在字符之后或者字符集之后，用来表示字符被匹配的次数

语法 说明 表达式实例 可以匹配到的字符串
* 匹配前一个字符0次或者无限次 abc* ab或者abcccc
+ 匹配前一个字符1次或者无限次 abc+ abc或者abcccc
? 匹配前一个字符0次或者1次 abc? ab或者abc
{m} 匹配前一个字符m次 a{2}c aac
{m,n} 匹配前一个字符m至n次（其中m或n可以忽略） a{1,2}c ac或者aac

边界匹配

这次字符主要表示设定开头，并不消耗字符

语法 说明 表达式实例 可以匹配到的字符串
^ 匹配字符串开头，在多行字符串中匹配字符串的开头 ^abc 匹配每行中abc开头的字符串
$ 匹配字符串的末尾，在多行字符串中匹配字符串的末尾 abc$ 匹配每行中abc结尾的字符串
\A 仅适用于单行模式，仅匹配字符串开头 \Awer wer开头的单行字符串
\Z 仅适用于单行模式，仅匹配字符串末尾 wer\Z wer结尾的单行字符串
\b 仅匹配\w和\W之间
\B [^\b]

逻辑，分组

这个代表了正则表达式的逻辑运算和分组策略

语法 说明 表达式实例 可以匹配到的字符串
\ 代表逻辑或，先尝试左边的表达式，如果尝试通过了直接返回，尝试未通过会继续尝试右边的表达式，并且支持()表示限定作用范围 abc(123\ 456)bcd abc123bcd或者abc456bcd
(…) 被括起来的表达式将作为一个分组，编号从1开始，从表达式的左边开始每遇到一个括号编号+1，另外分组作为一个整体后面依然可以接数量词，表达式中间的\ 的作用域将只在()内 abc(.*)def abc123def等等

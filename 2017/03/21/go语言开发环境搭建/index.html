go语言开发环境搭建

发表于 2017-03-21 | 分类于 golang ， 开发环境搭建 |

前言

golang 是云时代最值得学的一门语言，不仅语法简单，并且执行效率高效，工程部署方便，百说不如一撸

官网下载安装包（需要科学上网）

https://storage.googleapis.com/golang/go1.8.darwin-amd64.pkg

https://storage.googleapis.com/golang/ 这里面有你要的所有版本的下载地址

配置环境变量

来到根目录查看，编辑 .bash_profile 文件，配置环境变量并保存：

其中：

GOPATH: 日常开发的根目录
GOBIN: 是 GOPATH 下的 bin 目录，存放一些二进制文件

且需要将 gobin 目录添加到 path 路径下，生成的可执行文件就可以直接运行了

source 命令重新加载环境变量，使它生效 source ~/.bash_profile

最后 go env 命令就可以看到配置的路径了

IDE 推荐 jetbrains 的 GODLAND

jetbrains 的开发工具一直是我的最爱，下载地址: https://download.jetbrains.8686c.com/go/gogland-163.12024.32.dmg

配置 goland 开发环境

选择新建工程，默认会给你配置开发选项，当然也可以像 jetbrains 其他产品一样自己配置路径

你也可以自己使用命令行来运行这个程序 go run xxx.go，他会自己编译并运行

Go程序交叉编译

我们编写的这个 helloworld 程序现在只能在 macos 环境下运行，如果我们希望它运行在 Linux ，需要进行交叉编译，golang 的交叉编译非常简单，命令格式: CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build gofile.go

其中

GOOS：目标可执行程序运行操作系统，支持 darwin（macos），freebsd，linux，windows
GOARCH：目标可执行程序操作系统构架，包括 386，amd64，arm

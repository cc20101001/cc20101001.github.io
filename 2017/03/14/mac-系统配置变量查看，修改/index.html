mac 系统配置变量查看，修改

发表于 2017-03-14 | 分类于 mac |

前言

这篇文章记录了如何来修改macos一些内核参数

列出系统当前状态

命令: sysctl -a

列出的都是系统配置的一些属性

查看系统配置变量限制

命令: launchctl limit

该命令查询系统配置变量当前值和最大值，比如我们需要修改的是mac网络通信socket的最大连接数，即为macfiles

显示当前的各种用户进程限制

命令: ulimit -a

系统级别修改配置

命令: sysctl

修改当前用户limit

需要新建两个文件，一个是来设置用户能打开的最大文件描述符数目，一个来设置用户进程能打开的最大文件描述符数目

vim /Library/LaunchDaemons/limit.maxfiles.plist

vim /Library/LaunchDaemons/limit.maxproc.plist

第三步，配置文件权限，并重启系统，让配置生效:

发现 ulimit -a，命令下的配置生效了，到此配置成功。
